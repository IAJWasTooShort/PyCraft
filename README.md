## Installing

Download the latest [release](https://github.com/IAJWasTooShort/PyCraft/releases/latest), extract the zip, and run the exe.

#### Or run it locally

```console
$ git clone https://github.com/IAJWasTooShort/PyCraft.git
```
```console
$ cd PyCraft
```
```console
$ pip install -r requirements.txt
```
```console
$ python3 main.py
```
